一个简单的面试题

Given a collection of integer, design a structure that supports:
insert(data) - insert an element
delete(data) - delete an element
getRandom() - return a random element
all O(1) time

跟Cc的讨论内容：

我：vector + unordered_multimap

Cc：你的思路？

我：因为有random access的需要 所以我想到需要一个线性数据结构，就用vector吧。
然后需要线性时间insert，这个随便加到vector后面就行了，但是线性时间删除需要O(1)找到元素位置，
元素位置就放到unordered_map里面吧，但是怕有重复元素，就用multimap。
删除之后，为了维护vector的连续性，就把最后一个元素放到删除的位置，然后维护下map就行了。

Cc：明白。

我：这题我之前跟同学讨论过。而且，记得很牢。
coding应该也很简单，就是unordered_multimap会用就行了。
